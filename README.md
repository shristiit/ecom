# ecom

Monorepo layout:

- `apps/admin`: admin frontend
- `apps/backend`: Node/Express business API
- `apps/conversational-engine`: Python AI service
- `apps/ecommerce`: storefront/mobile app
- `apps/e2e`: end-to-end tests
- `packages/components`: shared UI component packages
- `infra/ops`: deployment and system configuration
- `docs`: repo-level documentation
